# task_1__elevate_labs__AIML

This repository contain Colab notebook to performs basic data preprocessing on the Titanic dataset available in the repository.

Steps included:
1. Import and explore the dataset (check data types, null values).
2. Handle missing values using mean/mode imputation.
3. Convert categorical features into numerical using encoding.
4. Normalize/standardize the numerical features.
5. Visualize outliers using boxplots and remove them.
6. The cleaned dataset is ready for further analysis or machine learning tasks.
